A modern recipe sharing platform built with Astro 5, React, and Convex. Share your culinary creations with the world.

**This is a complete rewrite of [RecipeSwap](https://github.com/dacrab/RecipeSwap)**, my original HTML/CSS/JS school project, now rebuilt as a full-stack application.

## Tech Stack

- **Framework:** [Astro 5](https://astro.build/) with React islands
- **Backend:** [Convex](https://convex.dev/) (Database, Auth, File Storage)
- **Deployment:** [Cloudflare Pages](https://pages.cloudflare.com/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)

## Features

- Recipe CRUD with image uploads
- Social interactions (likes, bookmarks, comments) with real-time updates
- Chef profiles and dashboards
- Category filtering and search
- Google OAuth + Email/Password authentication

## Quick Start

```bash
# Install dependencies
bun install

# Start Convex dev server (in one terminal)
bunx convex dev

# Start Astro dev server (in another terminal)
bun dev
```

## Environment Variables

```bash
# Convex (auto-generated by `convex dev`)
PUBLIC_CONVEX_URL="https://your-deployment.convex.cloud"

# Auth (set in Convex dashboard)
AUTH_GOOGLE_ID=""
AUTH_GOOGLE_SECRET=""
```

## Convex Setup

1. Run `bunx convex dev` to create a new project
2. Set up Google OAuth credentials in the [Convex dashboard](https://dashboard.convex.dev)
3. Add `AUTH_GOOGLE_ID` and `AUTH_GOOGLE_SECRET` as environment variables
